# Changelog

All notable changes to @stackloop/ui will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-03

### Added
- Initial release of @stackloop/ui
- 30+ React components with TypeScript support
- Framer Motion animations on all interactive components
- Single primary color theme system
- Comprehensive component library including:
  - **Form Components**: Button, Input, Textarea, Checkbox, Toggle, Slider, DualSlider, RadioPills, Dropdown, DatePicker
  - **Layout Components**: Card, Drawer, Modal, BottomSheet, Table, Pagination
  - **Display Components**: Badge, FloatingActionButton, StepProgress, ThumbnailGrid, StatusBadges, AudioRecorder, FileUpload
- ESM and CommonJS bundles
- TypeScript declaration files
- Theme CSS with design tokens
- Mobile-first responsive design
- Tree-shakeable exports

### Features
- Consistent border radius, shadows, and spacing across all components
- Smooth animations with Framer Motion
- Lucide React icons integration
- Touch-friendly interactions
- Accessibility considerations
- Loading states for async operations
- Error states and validation
- Customizable color variants

### Dependencies
- React >= 18.0.0 (peer dependency)
- React DOM >= 18.0.0 (peer dependency)
- Framer Motion 12.23.25
- Lucide React 0.555.0
- clsx 2.1.1
- tailwind-merge 2.6.0
